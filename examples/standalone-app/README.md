# Live example: standalone Angular app

This is a minimal standalone Angular example using `goey-toast-angular`.

## Run locally

```bash
npm install
npm run start
```

(If you're using this repo directly, copy these files into an Angular app generated with `ng new`.)

## StackBlitz starter

Use this template as a live playground:

- https://stackblitz.com/edit/angular-standalone-template

Then install the package:

```bash
npm install goey-toast-angular
```
